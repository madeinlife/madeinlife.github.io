web配置信息

配置文件是web项目根目录下的_config.yml,注意：:后面需要有个空格 ,否则会报错。

参数 描述
title 网站标题
subtitle 网站副标题
description 网站描述
author 您的名字
language 网站使用语言：eg:zh-CN
timezone 网站时区:eg:Asia/Shanghai
theme 网站使用模板名称:eg:jacman

添加新的文章
